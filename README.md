# StoryArc - Frontend

StoryArc is a creative production studio specializing in influencer shoots, commercials,
and digital content. This frontend project delivers a fast, modern, and visually refined
user experience for our clients and website visitors.

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`VITE_API_BASE_URL`

`VITE_API_USERNAME`

`VITE_API_PASSWORD`
